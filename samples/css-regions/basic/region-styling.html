← Text in this region should be red. Here's the CSS that achieves that:

@region #region1 {
    p {
        color: crimson;
    }
}

← Text in this region should be on a blue background. Here's the CSS that achieves that:

@region #region2 {
    p {
        background-color: lightblue;
    }
}
